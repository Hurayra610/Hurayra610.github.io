## Setup

### 1. **`git init`**

- Initialize a new Git repository in the current directory.

```bash
git init
```

### 2.**`git config`**

- Set user configuration options like username and email.

```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```
